# Overview
this is a collection of my scripts I (sometimes) run

## scripts
- aurman.sh: shell script to manage arch AUR packages
- extractimages.py: python script to extract jpgs out of images from failed sd-cards
- restorenames.sh: a shell script to fix the names generated by extractimages.py

## how to use
to add the script to your /usr/bin to be able to call it from anywhere execute in this repo 
```
sudo ln -s `pwd`/<script_name>.sh /usr/bin/<script_name>
```
